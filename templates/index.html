{% include "header.html" %} Welcome to Kieran's Magic: the Gathering game logger, for all your game tracking needs.

To get started, head to the Input page and create a player for yourself.
Once that is done, you can start adding your decks to the database.

The Data page lets you view tables of data.
Select "Colour Identity" to view the number of decks of each colour identity.
Select "Deck" to view information about each deck.
Select "Player" to view how many decks each player owns.

The Graphs page lets you view data in either bar chart or pie chart form. {% include "footer.html" %}
